# JX3 JJC 心法盲盒

一个抽到什么上什么的亲友整活小工具，也可帮助各位全职高手告别选择困难。

### 核心功能
- **多模式抽取**：支持个人单抽、队伍组队抽取、自定义选项抽取。
- **自定义配置**：可自由 Ban 掉不想要的心法，可指定队员固定为治疗位。
- **多主题切换**：内置长歌、明教、五毒 3 个视觉主题。
- **全平台适配**：完美适配 PC 端与移动端。

### 使用引导
1. 点击右下角 **齿轮图标** 进入配置。
2. 选择模式（个人/队伍/自定义）。
3. 根据需要筛选心法或设置队员名单。
4. 点击 **开始抽签**，愿赌服输。
